# Company Scenario

## Organization
ABC Healthcare

## Environment
- 500 employees
- 300 remote users
- hybrid workforce
- cloud-based applications

## Existing Problems
- VPN bottlenecks
- phishing attacks
- insecure internet access
- limited visibility into user web activity
- inconsistent remote access controls

## Proposed Solution
Deploy Zscaler Internet Access (ZIA) using SASE and Zero Trust principles to secure web traffic, enforce policy-based access, and centralize monitoring through Splunk.

## Security Goals
- secure remote internet access
- block phishing and malware domains
- inspect encrypted traffic
- improve visibility into user activity
- monitor threats using Splunk dashboards
