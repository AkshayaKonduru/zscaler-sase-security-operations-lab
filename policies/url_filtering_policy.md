# URL Filtering Policy

## Objective
Protect remote users from malicious and unauthorized internet access through Zscaler Internet Access (ZIA).

## Blocked Categories
- Phishing
- Malware
- Gambling
- Adult Content
- Suspicious Domains
- Cryptocurrency Mining

## Allowed Categories
- Business Applications
- Development Tools
- Education
- Healthcare Services
- Microsoft 365
- GitHub

## SSL Inspection Policy
Enable SSL inspection for unknown and risky domains to improve threat visibility and detect encrypted malicious traffic.

## User-Based Policies
- Finance users receive stricter filtering policies
- Developers allowed access to GitHub and technical resources
- Guest devices restricted from sensitive applications

## Logging and Monitoring
All web activity logs are forwarded to Splunk for centralized monitoring, dashboarding, and incident investigations.
